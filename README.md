# Dimitris.-L---MCDA---AHP-Python-Script
Αυτοματοποίηση Χωροθέτησης Αιολικού Πάρκου με χρήση GIS και Πολυκριτηριακής Ανάλυσης Αποφάσεων (MCDA). Η μεθοδολογία βασίζεται στην εργασία του Δημήτρη Λιούμη με τίτλο "Ανάλυση Καταλληλότητας για Χωροθέτηση Αιολικού Πάρκου με χρήση GIS και Πολυκριτηριακής Ανάλυσης Αποφάσεων (MCDA)" για το μάθημα "Εισαγωγή στην Επιστήμη των Γεωγραφικών Πληροφοριών" (Καθηγητής: Κ. Χαλκιάς) του ΔΠΜΣ Εφαρμοσμένη Γεωπληροφορική του Χαροκοπείου Πανεπιστημίου.
